## مسیرهای جامع یادگیری

### منابع بین‌المللی

- [OSSU Computer Science](https://github.com/ossu/computer-science)  
  مسیر جامع و خودآموز برای یادگیری علوم کامپیوتر با منابعی از دانشگاه‌هایی مثل MIT، Harvard و Princeton.[^https://github.com/ossu/computer-science]

- [Teach Yourself Computer Science](https://teachyourselfcs.com/)  
  یکی از بهترین نقشه‌راه‌ها برای یادگیری عمیق علوم کامپیوتر: الگوریتم، معماری، سیستم‌عامل، شبکه، دیتابیس، کامپایلر و سیستم‌های توزیع‌شده.

- [CS DIY](https://csdiy.wiki/en/)  
  نقشه‌راه بسیار کامل برای خودآموزی علوم کامپیوتر با دوره‌های دانشگاهی و پروژه‌های عملی.[^https://csdiy.wiki/en/CS%E5%AD%A6%E4%B9%A0%E8%A7%84%E5%88%92/]

- [Open Source Society University](https://github.com/ossu/computer-science)  
  مناسب برای کسانی که می‌خواهند معادل یک برنامه کارشناسی علوم کامپیوتر را خودآموز طی کنند.

- [Awesome Courses](https://github.com/prakhar1989/awesome-courses)  
  لیست بزرگ دوره‌های دانشگاهی علوم کامپیوتر از دانشگاه‌های معتبر دنیا.[^https://github.com/prakhar1989/awesome-courses]

- [CS Video Courses](https://github.com/Developer-Y/cs-video-courses)  
  مجموعه‌ای از ویدئوهای درسی دانشگاهی در زمینه الگوریتم، سیستم‌عامل، دیتابیس، AI، شبکه و غیره.[^https://github.com/Developer-Y/cs-video-courses]

- [Google Tech Dev Guide](https://techdevguide.withgoogle.com/)  
  راهنمای گوگل برای تقویت مهارت‌های فنی، حل مسئله و آمادگی شغلی.

- [freeCodeCamp](https://www.freecodecamp.org/)  
  مناسب برای یادگیری عملی برنامه‌نویسی، وب، بک‌اند، دیتابیس، Python، JavaScript و مسیرهای شغلی.

- [The Odin Project](https://www.theodinproject.com/)  
  مسیر پروژه‌محور برای توسعه وب Full-Stack.

- [Roadmap.sh](https://roadmap.sh/)  
  نقشه‌راه‌های تصویری برای Frontend، Backend، DevOps، AI، Cyber Security، Data Analyst و غیره.

## منابع فارسی / ایرانی

### پلتفرم‌های آموزشی فارسی

- [فرادرس](https://faradars.org/)  
  یکی از بزرگ‌ترین پلتفرم‌های آموزشی فارسی با دوره‌های دانشگاهی، برنامه‌نویسی، هوش مصنوعی، داده‌کاوی، لینوکس، شبکه و مهندسی کامپیوتر.[^https://faradars.org/]

- [مکتب‌خونه](https://maktabkhooneh.org/)  
  دوره‌های دانشگاهی و مهارتی، از جمله برنامه‌نویسی، هوش مصنوعی، داده‌کاوی و دوره‌های دانشگاه‌های ایرانی.[^https://maktabkhooneh.org/]

- [Quera College](https://college.quera.org/)  
  دوره‌های برنامه‌نویسی، الگوریتم، پایتون، جنگو، فرانت‌اند و آمادگی بازار کار.

- [Quera](https://quera.org/)  
  تمرین برنامه‌نویسی، مسابقه، آزمون استخدامی، حل مسئله و جامعه برنامه‌نویسان ایرانی.

- [جادی](https://jadi.net/)  
  مطالب، ویدئوها و دوره‌های مفید درباره لینوکس، پایتون، آزادی نرم‌افزار، شبکه و برنامه‌نویسی.

- [کانال یوتیوب جادی](https://www.youtube.com/@jadi)  
  آموزش لینوکس، پایتون، مفاهیم کامپیوتر و تجربه‌های فنی.

- [سبزلرن](https://sabzlearn.ir/)  
  دوره‌های فارسی توسعه وب، JavaScript، React، Node.js، Python و مهارت‌های بازار کار.

- [راکت](https://roocket.ir/)  
  آموزش فارسی برنامه‌نویسی، طراحی وب، JavaScript، PHP، Laravel و توسعه نرم‌افزار.

- [تاپ‌لرن](https://toplearn.com/)  
  دوره‌های فارسی برنامه‌نویسی، وب، موبایل، هوش مصنوعی و طراحی.

- [دانشجویار](https://www.daneshjooyar.com/)  
  دوره‌های فارسی برنامه‌نویسی، طراحی سایت، هوش مصنوعی و مهارت‌های نرم‌افزاری.

- [سون‌لرن](https://7learn.com/)  
  آموزش برنامه‌نویسی و طراحی وب.

- [کندو](https://cando.ac/)  
  آموزش‌های تخصصی IT، شبکه، امنیت، DevOps و زیرساخت.
